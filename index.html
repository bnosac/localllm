Welcome to localllm

Python package localllm provides functionalities to use a Local LLM object alongside dspy

Download a model

For the list of all models, look to the help of localllm_download_model

import pprint
from localllm import LocalLLM, LocalChatAdapter, localllm_download_model
path = localllm_download_model("gemma-3-270m-it-Q8_0")
path
'/home/runner/.localllm/models/google_gemma-3-270m-it-Q8_0.gguf'

Test the local model

from llama_cpp import Llama
transformer = Llama(model_path=path, n_gpu_layers=-1, flash_attn = False, n_ctx = 32768, n_threads = 1, seed = 4321, verbose = False)

Use the model directly alongside dspy

Use dspy alongside the local model

import dspy
dspy.configure(lm = LocalLLM(transformer), adapter = LocalChatAdapter())
class Go(dspy.Signature):
    sentence: str = dspy.InputField(desc = "A question")
    value: str    = dspy.OutputField(desc = "A city")

model = dspy.Predict(Go)
out = model(sentence="What is the capital of France")
pprint.pprint(out)
Prediction(
    value=None
)
out["value"]
